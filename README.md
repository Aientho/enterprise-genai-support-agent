# Enterprise GenAI Customer Support Agent

AI-powered customer support system using:

✅ GPT-4o

✅ LangChain

✅ LangGraph

✅ RAG

✅ ChromaDB

✅ FastAPI

✅ Human-in-the-Loop Workflow

## Business Use Case

Automates customer support requests by:

- Reading customer complaints
- Searching policy documents
- Generating responses
- Recommending refunds
- Escalating high-value refunds to managers
- Sending customer notifications
